## gitbook-plugin-gitlab

Display a link to your GitLab repo in your gitbook.

### Usage

Put this in your book.json:

```js
{
    "plugins": [ "gitlab" ],
    "pluginsConfig": {
        "gitlab": {
            "url": "https://gitlab.com/your/repo"
        }
    }
}
```

And you're done!
